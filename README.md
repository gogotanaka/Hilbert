```coffeescript
P | Q # P or Q
~P    # not P
Q?    # Q is TRUE?
```

### Differentiate


## How to use

Install hilbert gem.

    $ gem install hilbert

## Interpreter

    $ hilbert -i
    Enjoy! ->

## Use as native language

### Compile into R

    $ hilbert -r foo.hr

### Compile into Ruby

    $ hilbert -rb foo.hr

### Compile into Python

    $ hilbert -py foo.hr


## Use as math template within other langs
